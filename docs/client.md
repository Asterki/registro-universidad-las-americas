# Client (Web Frontend)

Location: `client/`

Overview

The frontend is implemented using Vite and React with TypeScript. It provides:

- Form pages for end-users to submit data and feedback
- Administrative pages to view aggregated reports and filters
- Localization support (see `client/src/locales`)

Local development

Install dependencies and run the dev server from `client/`:

```bash
cd client
pnpm install    # or npm install
pnpm dev        # or npm run dev
```

Build for production

```bash
cd client
pnpm build      # or npm run build
```
